# 🎮 Hangman Game – Word Guessing Game

## 🚀 Project Overview

This is a browser-based Hangman Game developed using HTML, CSS, and JavaScript.  
The game helps users improve vocabulary by guessing letters of a hidden word based on a given hint.

If the guessed letter is correct, it appears in the word.  
If the guess is incorrect, a part of the hangman figure is drawn.  
The game ends when the word is completed or the hangman structure is fully drawn.

---

## 🛠 Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## ✨ Features

- Random word selection
- Hint display for each word
- Letter-by-letter guessing
- Correct letter reveal in word placeholders
- Wrong answer tracking
- Progressive hangman drawing on incorrect guesses
- Game win/lose condition handling
- Restart game functionality

---

## 🎯 Game Logic

- A word is selected randomly from a predefined word list.
- The word is hidden with blank spaces.
- When the user selects a letter:
  - If correct → The letter appears in all matching positions.
  - If incorrect → A part of the hangman structure is drawn.
- The game ends when:
  - The player guesses the complete word (Win)
  - Maximum incorrect attempts are reached (Lose)
